# RedFox-Leukemia-Control

This repository provides the MATLAB/Simulink implementation of the optimized nonlinear control strategies for acute leukemia therapy presented in the manuscript:

**"RedFox Optimized Control Designs for Acute Leukemia Therapy"**

## Contents
- **Simulink Models**: SMC, ISTSMC, ATSMC, and IASTSMC controllers
- **MATLAB Scripts**: Dynamic leukemia model, optimization algorithm (RedFox), controller parameters
- **Figures**: Simulation results used in the manuscript
- **Documentation**: Instructions to reproduce the results

## How to Run
1. Open the desired `.slx` model in MATLAB R2024b or later
2. Load the parameters or run the setup script
3. Set simulation duration to 50 seconds (as referenced in the manuscript)
4. Run and analyze results
